# Aufgabenstellung

1. Regression

In diesem Projekt soll ein Regressionsmodell für Hauspreise implementiert werden. Das Modell soll auf Basis verschiedener Input-Daten den durchschnuttlichen Hauspreis in einem Distrikt (Wohngebiet mit 600 bis 3000 Einwohnern) berechnen.

2. Klassifikation

In der Klassifikation ist ein Modell zu implementieren, welches auf Basis verschiedener Input-Daten eine Vorhersagen trifft, ob der durchschnittliche Hauspreis in einem Distrik über bzw. unter einem bestimmten Wert (hier 150000 US-Dollar) liegt.
